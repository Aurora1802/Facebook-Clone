  1. When we place img inside a div (block) element, if we don't define the display: flex property for the div container => the img by default will follow the arrangement of the block (div).
  2. margin: bottom can be used to add more space between the items in the flex container.
  3. The unset CSS keyword resets a property to its inherited value if the property naturally inherits from its parent, and to its initial value if not.
  4. anchor (or link) is an inline element.
  5. Because a elements lie inside imp-links-block element, if we don't define the display property for its child, then the childs will have its layout-block, which mean they will display in vertical direction.
  6. flex: 1 0 auto => shorthand for flex-grow, flex-shrink, flex-basis. flex-basis sets the lenghts of the flex items. If a class has the property of flex as flex: 1; => each flex items inside this class will distributed available space equally among them. This ensures that all child elements expand to fill the container horizontally. More info: CSS flex property
  7. display: inline-block => Create a specific block for each element under its section to maintain the structure of each element.
  8. display: inline-flex => Does not reserve any specific space in normal form.
